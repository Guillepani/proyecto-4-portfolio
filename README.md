# Proyecto 4 - Portfolio

Portfolio personal desarrollado para el Proyecto 4 del bootcamp.

## Objetivo

Construir una web tipo portfolio siguiendo el diseño propuesto, aplicando una estructura por componentes, buenas prácticas y diseño responsive.

## Requisitos principales

- Maquetación responsive
- Uso de Vite
- Estructura basada en componentes
- Componentes reutilizables
- CSS separado por componente
- No incluir todo el contenido directamente en el HTML
- Añadir al menos un evento en JavaScript
- Mantener cada componente por debajo de 100 líneas de JavaScript

## Tecnologías

- HTML
- CSS
- JavaScript
- Vite
